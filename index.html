2048 游戏

游戏说明

这是一个基于 Pygame 开发的 2048 游戏应用程序。

由于这是一个桌面应用程序，无法直接在浏览器中运行。

如何运行游戏

  • 确保你的电脑上已安装 Python 3.9 或更高版本
  • 安装依赖：pip install -r requirements.txt
  • 运行游戏：python main.py

游戏操作

  • 鼠标操作：点击并拖动鼠标来移动方块
  • 键盘操作：也可以使用方向键（↑↓←→）来移动
  • 重新开始：按 R 键重新开始游戏

游戏规则

  • 通过滑动将相同数字的方块合并
  • 合并相同数字时，分数会增加
  • 目标是合成一个 2048 的方块
  • 当没有空位置且无法合并时，游戏结束

项目结构：

  • main.py - 主游戏入口
  • kernel/game_logic.py - 游戏核心逻辑
  • kernel/graphics.py - 游戏画面渲染
  • kernel/mouse_handler.py - 鼠标操作处理
  • requirements.txt - 依赖文件
